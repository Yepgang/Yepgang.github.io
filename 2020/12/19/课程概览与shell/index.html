课程概览与shell

Posted on 2020-12-19 | In 工程能力

关于shell

  • Shell俗称壳（用来区别于核），命令解析器，接收用户命令，调用相应的应用程序(怎么知道调用对应的程序??)
  • 常见的shell有bash、dash…dash主要目的是执行脚本而不是为了交互，速度快但是功能少于bash
  • #!/bin/sh:一般见于shell文件的第一行，用于指定解释该shell文件的sh路径
  • 如今Debian和Ubuntu中，/bin/sh默认已经指向dash，可以通过指令ls -l /bin/sh确认，通过sudo dpkg-reconfigure dash可以修改sh为bash，同理可切换bash为dash
  • /etc/shells包含支持的shell
  • 一个有意思的指令：echo 3 | sudo tee brightness

课后习题

  • 可以用echo和重定向符写一些简单的文件(平时用vim习惯了)
  • touch
    • 用于修改文件或者目录的时间属性，包括存取时间和更改时间。若文件不存在，系统会建立一个新的文件
    • a：访问时间/m：修改时间/c：如果该文件不存在则不创建
  • grep
    • 用于查找内容包含指定的范本样式的文件,如果不指定查找文件或者目录则从标准输入设备读取数据，找到返回值为0，找不到返回值为1
    • i:忽略大小写；C：上下文；v:反选
    • grep test *file
    • grep -r update /etc/acpi
  • chmod
    • User、Group、Other users对应ugo
    • rwx对应421
    • chmod ug+w,o-w file1.txt file2.txt
    • chmod a=rwx file
    • chmod 777 file
    • chmod -R a+r *：R表示递归
