初步想法

我想做大数据相关的研究，初步考虑的是以下两个方向。

  • 考虑到大数据技术体系目前已经趋于成熟，从技术体系上寻求创新和突破还是具有一定难度的，但是由于目前大数据相关技术尚处在落地应用的初期，所以有大量与行业落地相结合的创新点
  • 另外一个切入点是大数据分析。作为数据价值化的重要手段，大数据分析与机器学习有紧密的联系，所以大数据与机器学习相结合也是一个不错的选择

工作开展

  • 阅读感兴趣的论文
  • 阅读大数据领域的经典论文

目前初步打算阅读的论文包括四种类型:经典论文、大数据技术总结与探讨、大数据技术在其他领域的结合与创新、大数据与机器学习的结合。其中后三种我打算先看2017年相对高引的文章

  • part1：经典论文
    • MapReduce: Simplified Data Processing on Large Clusters
    • The Google File System
    • Bigtable: A Distributed Storage System for Structured Data
    • The Hadoop Distributed File System
    • In Search of an Understandable Consensus Algorithm
    • ZooKeeper: Wait-free coordination for Internet-scale systems
    • …
  • part2：大数据技术总结与探讨
    • Big Data technologies: A survey
      • 本文是对最近为大数据开发的技术进行调查的综述。它旨在帮助根据其技术需求和特定应用程序的需求选择和采用不同大数据技术的正确组合,对主要技术的特征，优势，局限性和用途进行了分类和讨论
    • Big data: Dimensions, evolution, impacts, and challenges
      • 本文介绍了大数据的集成视图，跟踪了大数据在过去20年中的发展，并讨论了处理各种结构化和非结构化数据必不可少的数据分析
    • A survey towards an integration of big data analytics to big insights for value-creation
      • 大数据分析（BDA）日益成为一种趋势实践，可以生成大量数据并提供有助于相关决策的新机会。本文概述了BDA的体系结构，此外，它还提供了对大数据分析在农业，医疗保健，网络安全和智慧城市等各种应用中使用情况的系统分析。本文还重点介绍了针对各种应用程序平台的大数据分析的先前研究，挑战，现状和未来方向
    • Social media big data analytics: A survey
      • 本文提出了对最近工作的回顾，以获得对社交媒体大数据分析研究主题的广阔视野。我们根据重要方面对文献进行分类。这项研究还比较了可能的大数据分析技术及其质量属性。此外，我们通过重点介绍各种研究的最新技术，方法和质量属性，讨论了社交媒体大数据分析的应用。还描述了大数据分析中的开放研究挑战
  • part3：大数据技术在其他领域的结合与创新
    • A review on the practice of big data analysis in agriculture
      • 尽管大数据分析正在推动各行各业的进步，但它尚未在农业中得到广泛应用。本文的目的是对农业的当前研究和研究工作进行回顾，以利用大数据分析的最新实践，以解决各种相关问题。本文强调了农业大数据分析向智能农业发展的巨大机会，表明大数据分析的硬件和软件，技术方法的可用性以及大数据源的日益开放将鼓励更多的学术研究农业部门的研究，公共部门计划和企业。这种做法仍处于早期发展阶段，需要克服许多障碍
    • Big data management in the mining industry
      • 采矿业面临着许多推动采用新技术的挑战。在信息和通信技术的不断进步的推动下，大数据是可以重塑整个采矿业前景的有前途的技术之一。尽管尝试了将大数据应用于采矿业的各种尝试，但是采矿业中大数据的基本问题（特别是大数据管理（BDM））仍然存在。本文旨在通过介绍BDM的基础知识来填补这一空白。这项工作简要介绍了大数据和BDM，并讨论了采矿业所面临的挑战，以表明实施大数据的必要性。它还总结了采矿业中的数据源，并提出了大数据对采矿业的潜在好处。这项工作还预见了建立全球数据库项目并将大数据与其他技术（即自动化）一起使用的未来，并得到了政府政策的支持并遵循国际标准。本文还概述了在采矿业中使用BDM的注意事项
    • Medical big data: promise and challenges
      • 大数据技术在医疗保健中具有许多应用领域，例如预测模型和临床决策支持，疾病或安全监视，公共卫生以及研究。
    • Big data in tourism research: A literature review
      • 本文可能是对旅游研究中不同类型的大数据进行全面文献综述的首次尝试
    • Analysis of healthcare big data
      • 探讨大数据背景下中国医疗保健的发展以及医疗数据中的隐私和安全风险因素，分析了中国医疗保健行业的发展状况。该问卷用于分析医疗大数据的隐私和安全风险因素，并在文献中基于云服务的背景下根据数据隐私和安全风险因素提出了保护措施
  • part4：大数据与机器学习的结合
    • Machine learning on big data: Opportunities and challenges
      • 在本文中，我们介绍了基于大数据的机器学习框架（MLBiD），以指导讨论其机遇和挑战。该框架以ML为中心，它遵循预处理，学习和评估的阶段。此外，该框架还由其他四个组件组成，即大数据，用户，域和系统。 ML的阶段和MLBiD的组成部分为识别相关的机会和挑战提供了方向，并为许多未开发或未开发的工作打开了未来的工作
    • A survey on deep learning for big data
      • 在本文中，我们回顾了用于大数据特征学习的深度学习模型的新兴研究。此外，我们指出了大数据深度学习的剩余挑战，并讨论了未来的主题
    • Deep Learning for IoT Big Data and Streaming Analytics: A Survey
    • How can Big Data and machine learning benefit environment and water management: a survey of methods, applications, and future directions
    • A survey of big data architectures and machine learning algorithms in healthcare
    • A survey of machine learning algorithms for big data analytics
